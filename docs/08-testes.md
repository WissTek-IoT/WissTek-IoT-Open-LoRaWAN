# Testes Realizados

Foram realizados testes práticos com duas placas ESP32 LoRa:

- **Modo:** ponto-a-ponto
- **Alcance:** ~500 metros em área urbana sem obstáculos diretos

- **Taxa de envio:** ~1 pacote/segundo
